# 音效文件说明

本目录用于存放游戏的音效文件。

## 需要的音效文件

请将以下音效文件放置在此目录下：

1. **draw-card.mp3** - 发牌音效（卡牌从牌库到手牌的声音）
2. **play-card.mp3** - 出牌音效（打出卡牌的声音）
3. **attack.mp3** - 攻击音效（角色攻击的声音）
4. **hit.mp3** - 受击音效（受到攻击的声音）
5. **heal.mp3** - 治疗音效（治疗恢复的声音）
6. **victory.mp3** - 胜利音效（战斗胜利的声音）
7. **defeat.mp3** - 失败音效（战斗失败的声音）

## 音效要求

- 格式：MP3（推荐）或 WAV
- 时长：建议 0.5-2 秒
- 音量：建议标准化处理，避免过大或过小
- 文件大小：建议每个文件 < 100KB

## 临时方案

如果暂时没有音效文件，系统会静默运行，不会报错。你可以：
1. 使用在线音效库（如 freesound.org）下载免费音效
2. 使用音频编辑软件（如 Audacity）创建简单音效
3. 暂时跳过音效，系统会正常工作

## 注意事项

- 音效文件需要放在 `public/sounds/` 目录下
- 文件名必须与 `soundManager.ts` 中配置的路径一致
- 如果文件不存在，系统会静默处理，不会影响游戏运行
